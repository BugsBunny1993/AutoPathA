AutoPathA
=========
这是一个A*寻道算法的实现，由Kowalski QQ：547098469编写， 希望大家多提意见

运行在其他平台下比如安卓，iOS，Win32需要设置分辨率为1136*640， Mac-64bit平台下不需要设置，因为我就是在这个平台上做的.

其他信息：
使用方法类似与下面连接地址的项目

开发环境：

cocos2dx 2.2.1 
apple Mac OS X 10.9.1 Mavericks
Xcode 5.0.2 
语言：C++
